IntelliJ IDEA Plugin Template (Without AI)

这是一个 IntelliJ IDEA 插件开发模板，提供了快速创建新插件的基础框架和最佳实践。

模板特性

  • 🚀 快速开始：提供完整的项目结构和配置，开箱即用
  • 📦 依赖管理：预配置常用依赖，包括 Lombok、JUnit、Mockito 等
  • 🔧 开发工具：集成 Google Java Format、测试框架和插件验证工具
  • 🌍 国际化：内置多语言支持，提供中文和英文资源文件
  • 📝 文档完善：提供详细的使用文档和开发指南
  • 🎨 图标管理：提供图标管理类，统一管理插件图标资源

开发环境

  • JDK 17+
  • Gradle 8.0+
  • IntelliJ IDEA 2022.3+

快速开始

  1. 📥 克隆模板：克隆模板项目到本地
  2. ⚙️ 配置信息：修改 gradle.properties 中的插件信息
  3. ▶️ 启动开发：运行 ./gradlew runIde 启动沙箱环境
  4. 💻 开始开发：开始开发你的插件功能

包含功能

  • 🎯 Action 示例：右键菜单触发操作示例
  • 🎨 图标管理：统一管理图标资源的工具类
  • 🔔 通知工具：统一的通知功能工具类
  • 🌍 国际化支持：多语言资源文件支持

技术栈

  • 基于 IntelliJ Platform SDK 2.1.0
  • 使用 Kotlin DSL 配置构建
  • 集成 JUnit 5 测试框架
  • 支持 Mockito 进行单元测试
